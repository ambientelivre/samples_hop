# Apache Hop Samples

Examples and resources for the Apache Hop.

This repository contains sample pipelines, datasets, presentations, and test materials for Apache Hop.

## Repository Structure

datasets/
Sample datasets.

pipelines/
Example of features and options in Apache Hop.

presentations/
Materials for presentations like TDC.

unit_test/
unit tests in Apache Hop.

## Example

Example pipeline:

pipelines/HOP_EMPTY_STRING_DIFFERS_FROM_NULL/

This example demonstrates how Apache Hop handles the difference between empty strings and NULL values.
